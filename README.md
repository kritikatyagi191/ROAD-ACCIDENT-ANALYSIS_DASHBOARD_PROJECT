# ROAD-ACCIDENT-ANALYSIS_DASHBOARD_PROJECT
This Power BI dashboard provides an interactive analysis of road accident data to identify key patterns and risk factors.
It highlights overall casualties, fatalities, and total vehicles involved using KPI visuals.
The dashboard analyzes accidents based on severity levels to understand impact intensity.
 Accidents are examined by vehicle type, showing which vehicles are most involved.
Alcohol-related accidents are separately visualized for safety assessment.
 Speed limit analysis helps identify accident-prone speed zones.
Time-based insights include accidents by day of week, month, and year.
Location-based analysis covers urban, rural, highways, and national roads.
The dashboard enables easy filtering and comparison across multiple dimensions
